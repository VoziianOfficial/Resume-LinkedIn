      Essential Preference Memory

      This Cookie Policy explains how NimoMark may use cookies, local browser storage and similar technologies when visitors use the website.

      Section 01

      Policy Overview

      Cookies and similar browser-storage technologies can help a website remember information between page visits.

      NimoMark may use limited storage to maintain essential website behavior, remember consent choices and protect inquiry functionality against misuse.

      Essential website functionality

      Essential storage may be used without optional advertising or profiling cookies when it is reasonably necessary for security, consent management or website operation.

      The use of third-party tools may change over time. When a tool introduces non-essential storage, consent controls and this policy should be updated before that technology is enabled.

      Section 02

      Cookie Basics

      A cookie is a small piece of information that a website may ask a browser to store. Cookies can remain only for a browser session or continue until they expire or are removed.

      Local storage is a related browser technology that allows a website to save limited information directly in the browser without using a traditional cookie.

      Session cookie
      Usually removed when the browser session ends.
      Persistent cookie
      Remains until its configured expiration date or until the visitor removes it.
      First-party storage
      Created by the website that the visitor is currently using.
      Third-party storage
      Created by an external service loaded through the website.
      Local storage
      Browser storage that can remember preferences until the visitor clears or replaces them.
      Section 03

      Essential Storage

      Essential storage supports functionality that is reasonably necessary for the website to operate securely and remember basic visitor choices.

      Cookie consent preference

      Essential

      Remembers whether the visitor accepted all currently permitted preferences or selected essential storage only.

      Storage key: nimomark_cookie_consent_v1

      Storage type: browser local storage.

      Security and request controls

      Essential

      Hosting or server systems may temporarily process request information to detect spam, excessive submissions, automated misuse and technical security events.

      This processing may involve IP address, request timing and related technical metadata.

      Interface state

      Functional

      Temporary page state may be used while a visitor opens menus, accordions or other interface controls.

      This state normally remains only during the current page session unless a specific preference is intentionally stored.

      Blocking essential storage may prevent the website from remembering consent choices and can affect certain security or interface behavior.

      Section 05

      Third-Party Tools

      The website may load resources from third-party providers, including font, icon, animation, hosting or content-delivery services.

      A third-party provider may receive technical request information such as IP address, browser information, referring page and request time when its resource is loaded.

      Current third-party resources may include:

      • Google Fonts for website typography.
      • Content-delivery networks used for Lucide icons, AOS animations or Swiper assets.
      • Website hosting, email delivery and server infrastructure.

      External services have separate policies

      NimoMark does not control the storage, retention or security practices of an external provider. Visitors should review the provider’s own privacy and cookie information where relevant.

      Advertising, behavioral profiling or cross-website tracking technologies should not be introduced without appropriate disclosure, configuration and consent controls.

      Section 06

      Managing Preferences

      Visitors can manage the NimoMark cookie preference by selecting one of the options on this page or using the website cookie banner.

      1. Select “Accept permitted preferences” to save an accepted status in the browser.
      2. Select “Essential storage only” to limit the saved preference to essential functionality.
      3. Return to this page at any time to replace the current preference.
      4. Clear the website’s local storage through the browser to remove the saved choice completely.

      Removing the saved preference may cause the cookie banner to appear again during a later visit.

      A preference saved in one browser or device does not automatically transfer to another browser, device or private-browsing session.

      Section 07

      Browser Controls

      Most browsers allow visitors to review, remove or block cookies and local storage through privacy or website-data settings.

      Browser controls commonly allow a visitor to:

      • View stored website data.
      • Delete information saved by a specific website.
      • Block third-party cookies.
      • Block all cookies or local storage.
      • Clear website data automatically when the browser closes.
      • Use private or incognito browsing modes.

      Browser settings differ between products and versions. Visitors should use the help documentation provided by their browser developer.

      Blocking storage may affect consent persistence, contact-form protections or other website functionality.

      Section 08

      Storage Retention

      The cookie-consent preference remains in browser local storage until it is replaced, removed by the visitor or cleared by browser settings.

      Temporary technical information used for website security or request management may be retained for a limited period determined by the relevant hosting, server or infrastructure provider.

      Third-party resources may apply their own retention periods. Those periods are governed by the third party’s settings and policies.

      You can remove the preference at any time

      Clearing website data in the browser removes the locally stored NimoMark consent preference from that browser.
